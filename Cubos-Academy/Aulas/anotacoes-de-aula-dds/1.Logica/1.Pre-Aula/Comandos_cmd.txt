# Comandos CLI

	- ls 			        [lista os arquivos no diretório]
	- cd 'nome da pasta' 	[abre o diretório 'nome da pasta']
	- cd ..			        [volta ao diretório anterior]
	- cls				    [limpa o terminal]
	- tecla 'TAB' 		    [completa o nome do diretório]
	- dir_01 \ dir_02	    [abre tiratamente p segundo diretório sem precisar abrir um à um]
	- code .			    [abre o vscode no diretório atual]
	- mkdir <file>		    [cria um diretório com nome <file>]
	- echo >> <file>		[cria um aquivo no diretório de nome <file>]
	- rm <file>			    [apaga o arquivo de nome <file>]
	- rmdir <file>	    	[apga o diretório de nome <file>]
	- pwd					[mostra o path]
	- cp 					[copia um arquivo ou diretório]
	- mv					[move um arquivo ou diretório]
	- grep					[pesquisa por uma determinada string em um arquivo ou diretório]
	- chmod					[muda as permissões de um arquivo ou diretório]
	- sudo 					[executa um comando como superusuário]
	- ssh					[conecta-se a um servidor remoto via SSH]
	- scp					[copia arquivos de/para um servidor remoto via SSH]
	- tar					[cria e extrai arquivos compactados]
	- curl					[faz solicitações HTTP e exibe a resposta no terminal]
	- ps 					[exibe processos em execução]
	- top					[exibe as principais informações sobre o sistema e seus processos em execução]
	- tree 					[exibe a árvore de arquivos no diretório atual]
